## 📖 About

**Apple_Slice** is a clipboard history application that lets you store and access previously copied content across your system.

It features a clean and minimal interface and was built for fun while exploring desktop app development using modern web technologies.

The goal of this project is to create a fast, simple, and OS-friendly clipboard manager that works on any operating system supported by Electron.

---

## ✨ Features

* 📋 Automatically stores clipboard history
* 🧾 Simple and minimal UI
* 💻 Cross-platform (Windows, macOS, Linux)

### 🚧 Upcoming Features

* 🖼 Copy and store images
* 🔍 Search through clipboard history

## ▶️ Usage

1. Run the app
2. Copy text anywhere on your system
3. Open Apple_Slice to view your clipboard history using ctrl + shift + v
4. Click an item to copy it again
